## Workflow 1: Main Chatbot

### `workflow-main-chatbot-v5.json` (51 нод)

Основной workflow бота — реагирует на сообщения пользователей:

- Приём сообщений из Telegram (текст + фото)
- Онбординг пользователей (9 стадий)
- Генерация стратегии подготовки (периодизация по фазам)
- Генерация недельных планов
- Распознавание фото тренировок (GPT-4o Vision)
- Объединение данных из 2 скриншотов
- Прогресс за неделю после каждой тренировки
- Защита от дубликатов тренировок
- Сохранение истории чатов и всех данных в Supabase

**Workflow ID:** `7Ar459SadzSXgUEv`

## Workflow 2: Weekly Summary

### `workflow-weekly-summary.json` (12 нод)

Scheduled workflow — запускается автоматически каждое воскресенье в 20:00 МСК:

- Собирает итоги недели для каждого активного пользователя
- Отправляет AI-оценку прошедшей недели
- Автоматически генерирует план на следующую неделю
- Сохраняет новый план в БД, деактивирует старый

**Workflow ID:** `L6btLLT88hsIW9Eh`
